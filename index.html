تعريف n8n

n8n — منصة الأتمتة التي تجمع آلاف التطبيقات في مكان واحد
n8n هو برنامج أتمتة سير العمل مفتوح المصدر يسمح بربط أي تطبيقين وأتمتة المهام بينهما بدون كود.

المصدر: docs.n8n.io

المزايا الرئيسية

  • مفتوح المصدر: استضافة ذاتية مجانية على سيرفرك
  • +400 تطبيق: Google Sheets، Slack، WhatsApp، OpenAI
  • واجهة بصرية: اسحب وأفلت لبناء سير العمل
  • دعم AI متكامل: GPT، Claude، Gemini، Mistral

ما هي العقدة؟

الشبكات المترابطة — كيف تعمل العقد في n8n
العقدة (Node) هي وحدة عمل مستقلة تمثّل عملية أو تطبيقاً واحداً. كل عقدة تستقبل بيانات، تعالجها، ثم ترسلها للعقدة التالية.

المحفزات (Triggers)

كيف تبدأ سير العمل — الأحداث التي تطلق الأتمتة

المحفز (Trigger) هو العقدة الأولى في أي سير عمل. وظيفتها: الانتظار حتى يحدث شيء معين ثم إطلاق بقية سير العمل.

التعبيرات (Expressions)

استخدام البيانات الديناميكية لجعل سير العمل أذكى

ما هي التعبيرات؟

التعبير (Expression) هو كود JavaScript مختصر يُكتب داخل حقل العقدة لجعل قيمته ديناميكية تتغير بناءً على بيانات العقدة السابقة.
بدلاً من كتابة "مرحباً محمد" يدوياً، تكتب تعبيراً فيصبح الاسم تلقائياً من البيانات القادمة.

أمثلة على التعبيرات

الوصول للبيانات
{{ $json.name }}
{{ $json.email }}
{{ $json.data.price }}
الوصول لحقل في البيانات
التاريخ والوقت
{{ $now.toISO() }}
{{ $now.format('yyyy-MM-dd') }}
{{ $today }}
التاريخ والوقت الحالي
من عقدة سابقة
{{ $('Sheets').item.json.name }}
{{ $('Webhook').first().json.id }}
بيانات أي عقدة بالاسم
عمليات بسيطة
{{ $json.price * 1.15 }}
{{ $json.name.toUpperCase() }}
{{ $json.items.length }}

الدرس الثامن

الاعتمادات (Credentials)

ربط التطبيقات الخارجية بشكل آمن

ما هي الاعتمادات؟

الاعتمادات (Credentials) هي معلومات المصادقة (API Keys، OAuth) التي تحتاجها n8n للتواصل مع التطبيقات الخارجية بشكل آمن.
الأمان الرقمي — حماية بيانات الاعتماد في n8n

أنواع الاعتمادات

API Key

مفتاح فريد لخدمة خارجية.

OpenAI، Weather

OAuth2

تسجيل دخول آمن للخدمات الكبيرة.

Google Sheets، Gmail

Username/Password

بيانات دخول تقليدية.

Email SMTP، SSH

كيف تضيف Credential؟

1
افتح قائمة Credentials من الشريط الجانبي في n8n
2
اضغط "Add Credential" وابحث عن التطبيق
3
أدخل المعلومات المطلوبة واضغط "Save"
4
اختر هذا الاعتماد من أي عقدة تطلبه

تنبيه: لا تشارك ملفات n8n التي تحتوي على Credentials. استخدم Environment Variables في الإنتاج.

المصدر: docs.n8n.io/credentials
